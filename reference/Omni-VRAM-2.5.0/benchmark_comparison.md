### Omni-VRAM Architecture Advantages

1. **Zero-Copy CUDA Pipeline**: VRAM 内直注入，省去 CPU↔GPU 拷贝开销
2. **Fused Audio Frontend**: VAD + pre-emphasis + windowing 单 kernel 完成
3. **Streaming-First Design**: 滑动窗口 VAD + 增量转录，首 token 延迟更低
4. **Multi-Backend Support**: faster-whisper / whisper.cpp / OpenAI API / Distil-Whisper 统一接口
5. **Production Ready**: gRPC + REST + WebSocket 多协议，Prometheus 监控，插件系统

### Expected Performance Characteristics

| Scenario | Expected Behavior |
|----------|-------------------|
| Short audio (10s) | Omni-VRAM 与 faster-whisper RTF 接近（模型推理主导） |
| Long audio (60s+) | Omni-VRAM 预处理优势显现，RTF 更低 |
| Streaming latency | Omni-VRAM 首 token 更快（增量处理 vs 全量等待） |
| VRAM usage | Omni-VRAM 预分配策略，VRAM 峰值更可控 |
